## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation NPM

```bash
$ npm install
```

## Running the app NPM

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Buil Docker

```bash
$ docker-compose build
```

## Run Docker

```bash
$ docker-compose up
```

## Technologies Used

- NodeJS
- NestJS
- PostgreSQL
- Redis
- RabbitMQ
- Socket
- AWS
- MailJet
- JWT

## Modules

```
 The auth module is the module where the authentication and authorization processes of the users are made.
```
```
The user module is the module where the operations related to the users of the users are made.
```
```
The event module is a module that allows users to perform operations related to events.
```
```
The cardExtra module It is the module where users perform operations related to credit card extracts in the events they create.
```
```
The clue module is the module that allows users to take action on the clues of the events they create.
```
```
The expression module is the module that allows users to take action on the expressions of the events they create.
```
```
The message module is the module that allows users to take action on the messages of the events they create.
```
```
The team module is the module where users can perform operations related to teams.
```
```
The report module is a module that allows users to retrieve the data they want from the events they want.
```
